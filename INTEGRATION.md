## как использовать

### 1. Активация плагина
1. Зайдите в WordPress Admin
2. Перейдите в Плагины
3. Активируйте "Site Optimizer Pro"

### 2. Панель управления
Перейдите в **Site Optimizer → Dashboard** для:
- Запуска оптимизации изображений
- Оптимизации базы данных
- Просмотра статуса здоровья
- Статистики

### 3. Настройки
Перейдите в **Site Optimizer → Settings** для настройки:
- Создание WebP (вкл/выкл)
- Качество сжатия (60-100%)
- Авто-оптимизация (ежедневно)
- Очистка при активации

### 4. Виджет
1. Внешний вид → Виджеты
2. Добавьте "Site Optimizer - Здоровье сайта" в Sidebar
3. Настройте заголовок и опции

### 5. Shortcodes
Вставьте shortcodes в любые записи/страницы для отображения статуса

## требования

- WordPress 5.0+
- PHP 7.0+ (рекомендуется 7.4+)
- GD или Imagick extension
- MySQL 5.6+
- Тема Hacker (или совместимая)

## поддержка

При возникновении проблем:
1. Проверьте требования PHP
2. Убедитесь, что GD/Imagick установлен
3. Проверьте права на запись в wp-content/uploads
4. Включите WP_DEBUG для отладки
